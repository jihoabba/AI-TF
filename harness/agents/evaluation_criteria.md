# 평가 기준표 — LINE FRIENDS SQUARE AX TF

Generator와 Evaluator가 공유하는 품질 기준.
이 하네스는 AX TF 아이디어를 목업·프로토타입으로 구체화하는 용도로 사용된다.

---

## 채점 항목

### 1. 비즈니스 적합성 (비중: 40%)
이 목업이 실제 현업에서 쓰일 수 있는가?

합격: 명확한 Pain Point 한 개 이상 해결, 특정 팀/담당자의 반복 업무를 줄여줌, 월 10시간 이상 절감 가능성 보임, TF 착수 기준 충족
불합격: 어느 팀이 왜 쓰는지 불명확, 기존 툴(엑셀, 노션 등)로 충분히 해결 가능, 일회성·전시용 느낌

TF 착수 기준 (이 중 2개 이상 충족해야 합격):
- 월 10시간 이상 절감 예상
- 2개 팀 이상 공통 적용 가능
- 반복성이 명확하고 자동화 효과가 큼
- Track A(경영진 의사결정) 또는 Track B(실무 반복 제거) 중 하나에 명확히 속함

### 2. 사용성 (비중: 30%)
사람이 자주 쓰게 만드는가? — TF 핵심 철학

합격: 처음 보는 실무자가 설명 없이 3분 안에 쓸 수 있음, 주요 액션이 화면에 바로 보임, 불필요한 단계 없음
불합격: 기능은 있지만 어디 있는지 모름, 클릭 경로가 3단계 이상, 실무에서 안 쓸 것 같은 느낌

### 3. 기술적 완성도 (비중: 15%)
기본기가 갖춰져 있는가?

합격: 반응형 정상 작동, 글자/간격 일관, 에러 없음, 데이터 흐름이 논리적으로 구현됨
불합격: 레이아웃 깨짐, 버튼 동작 안 함, 콘솔 에러

### 4. 디자인 품질 (비중: 15%)
LINE FRIENDS SQUARE 내부 도구로서 완성도가 있는가?

합격: 일관된 색상 팔레트, 깔끔한 여백, AX TF 아카이브 사이트(Apple 감성 다크/라이트)와 어울림
불합격: AI slop 패턴(보라 그라데이션 + 흰 카드), 색상 산만, 섹션마다 다른 느낌

---

## 판정

가중 점수 = (비즈니스적합성×0.4) + (사용성×0.3) + (기술×0.15) + (디자인×0.15)

- 7.0 이상 → 합격
- 5.0~6.9 → 조건부 합격 (피드백 반영 후 재검수)
- 5.0 미만 → 불합격
- 비즈니스 적합성 또는 사용성 4점 이하 → 무조건 불합격

---
