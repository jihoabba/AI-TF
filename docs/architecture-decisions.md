# AI TF 아키텍처 의사결정 기록

---

## 2026-04-02 · AI DESK 데스크톱 앱 + 맥미니 로컬 서버 아키텍처

### 배경

- 인터뷰 결과, 여러 팀이 거의 동일한 SAP 엑셀 데이터를 각자 다운받아 사용 중
- 각 팀 담당자가 복잡한 엑셀 수식으로 가공 → 담당자만 이해 가능한 블랙박스 상태
- 매일/매주 정해진 시간에 담당자가 수동으로 Box에 공유하는 워크플로우

### 결정 사항

**1. AI DESK를 Electron 데스크톱 앱으로 제작**
- 팀별 캐릭터 AI 에이전트 GUI
- B2B 매출 리포트, 발주 카탈로그 기능 LIVE 연동 완료
- 위치: `ai-desk-app/`

**2. 백엔드는 맥미니 로컬 서버로 운영**

```
맥미니 (사무실 상시 가동)
├── Box Drive → SAP 엑셀 자동 동기화
├── 파서 (Node.js) → 엑셀 변경 감지 → SQLite 저장
├── API 서버 (Express) → 각 팀에 필요한 데이터 제공
└── Claude API 연동 → 자연어 질의 응답
```

**3. SAP 데이터는 외부 전송 금지**
- Supabase 등 외부 DB/클라우드 사용 불가 (보안)
- Claude API 호출 시 원본이 아닌 **집계된 숫자만** 전송
- 데이터는 사내 네트워크(같은 와이파이) 내에서만 이동

### 각 팀이 같은 데이터에서 받는 것

| 팀 | 데이터 | 제공 형태 |
|---|---|---|
| B2B | 판매처별 매출, 공헌이익 | 매출 리포트 + 자연어 요약 |
| MD | SKU별 재고, 소진율 | 발주 추천 + 이관 알림 |
| 커머스 | 채널별 셀스루 | 대시보드 + 트렌드 분석 |
| 오프라인 | 매장별 일별 판매 | 일일 리포트 자동 생성 |

### 필요 리소스

- 맥미니 1대 (M시리즈)
- Box Drive (기존 사용 중)
- Claude API 키
- 사내 와이파이 접근

### 기대 효과

- 엑셀 수식 블랙박스 해소 → 담당자 부재 시에도 데이터 자동 처리
- 팀별 수작업 리포트 생성 → 자동화
- Claude API를 통한 자연어 데이터 질의 가능
